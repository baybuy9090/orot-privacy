핵심 요약
오롯은 사용자의 데이터를 외부 서버로 전송하거나 수집하지 않습니다.
모든 데이터는 사용자의 기기 안에서만 저장되고 관리됩니다.

개발자(이하 "운영자")는 오롯(OROT) 앱 운영과 관련하여 개인정보보호법 등 관련 법령을 준수하며, 이용자의 개인정보를 안전하게 보호합니다.

1. 수집하는 정보

오롯은 별도의 회원가입이나 로그인 없이 사용하는 앱입니다. 운영자가 직접 수집하는 개인정보는 없습니다.

앱 이용 과정에서 사용자가 직접 입력하는 아래 데이터는 기기 내부에만 저장됩니다.

  • 활동 기록 (카테고리, 시간, 메모, 만족도)
  • 계획한 세션 정보 (시작/종료 시간, 카테고리, 상세 내용)
  • 주간 목표 및 인생 목표 텍스트
  • 자유 시간 설정값
  • 타이머 중 촬영한 사진 (기기 내 앱 전용 폴더에만 저장)

이 데이터는 외부 서버로 전송되지 않으며, 앱 삭제 시 모두 삭제됩니다.

2. 수집하지 않는 정보

오롯(운영자)은 다음 정보를 직접 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 식별 가능한 개인정보
  • 연락처 등 기기 내 민감한 데이터

단, 광고 표시를 위해 Google AdMob이 광고 식별자(IDFA) 등을 수집할 수 있습니다. (4항 참고)

3. 권한 사용

앱 기능 제공을 위해 아래 기기 권한을 사용할 수 있습니다.

  • 알림(Notifications) — 일일 오롯 리마인더 및 계획한 활동 시작 전 알림 발송
  • 오디오(Audio) — 타이머 완료 효과음 재생
  • 카메라 / 사진 보관함(Camera / Photo Library) — 사용자가 타이머 중 순간을 기록하기 위해 직접 촬영하거나 선택한 사진을 앱 내 세션에 저장합니다. 사진은 기기 내 앱 전용 폴더에만 보관되며 외부로 전송되지 않습니다.
  • 캘린더(Calendar) — 사용자가 허용한 경우, 아이폰 캘린더 일정을 플래너에 표시하고 오롯에서 추가한 계획을 캘린더에 등록합니다. 캘린더 데이터는 외부 서버로 전송되지 않으며 기기 내에서만 처리됩니다.
  • 위치(Location) — 사용자가 허용한 경우, 위치 기반 리마인더 기능에 사용됩니다. 위치 데이터는 기기 내에서만 처리되며 외부 서버로 전송되지 않습니다.
  • Apple 건강(HealthKit) — 사용자가 허용한 경우, Apple 건강 앱의 운동 기록을 읽어 오롯타임에 자동으로 포함합니다. 건강 데이터는 외부 서버로 전송되지 않으며 기기 내에서만 처리됩니다. 오롯은 건강 데이터를 기록하거나 수정하지 않습니다.
  • 스크린 타임(Screen Time / Family Controls) — 사용자가 허용한 경우, 타이머 집중 시간 동안 선택한 앱을 차단합니다. 차단 설정은 기기 내에서만 처리되며 외부로 전송되지 않습니다. 타이머 종료 시 즉시 차단이 해제됩니다.

권한은 사용자가 언제든지 기기 설정에서 변경하거나 거부할 수 있습니다.

4. 제3자 서비스

광고 — Google AdMob

무료 사용자에게 광고를 표시하기 위해 Google AdMob을 사용합니다. AdMob은 광고 개인화 목적으로 광고 식별자(IDFA) 및 기기 정보를 수집할 수 있습니다. 오롯 PRO를 구매하면 광고가 완전히 제거됩니다.

  • Google 개인정보처리방침

인앱 결제 — Apple StoreKit

오롯 PRO 구매는 Apple의 StoreKit을 통해 처리됩니다. 결제 정보(카드 번호 등)는 Apple이 직접 관리하며, 운영자는 결제 세부 정보에 접근하지 않습니다.

  • Apple 개인정보처리방침

콘텐츠 공유

앱 내 '공유' 기능은 iOS 시스템의 공유 시트를 통해 사용자가 직접 선택한 앱으로 데이터를 전달합니다. 이 과정에 운영자는 관여하지 않습니다.

5. 데이터 보관 및 삭제

앱에 저장된 모든 데이터는 사용자의 기기에만 보관됩니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

운영자는 사용자 데이터에 접근하거나 삭제를 요청받을 권한이 없습니다.

6. 아동 개인정보

오롯은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

개인정보처리방침이 변경될 경우, 앱 업데이트 또는 본 페이지를 통해 사전에 공지합니다.
